Network Administrator/Analyst-AAS Degree

As stated on the Hennepin Tech website; the Network Administrator degree provides the skills needed to design, manage, troubleshoot and secure a network environment. I chose this program mainly because I have been around computers for a long time, whether it be in a home, school, or work setting. I think with technology progressing it is important to know how to build, maintain, and troubleshoot a computer. Also it is very important to know how to keep things secure as possible.
